Gestión llaves

1. Creando llaves en el HSM

(detalles)
En este modulo "you will be using key_mgmt_util software provided in cloudHSM Client package to create symmmetric keys, RSA Key pair and ECC key pair."

  • 1.1. Vuelva a la pestaña o ventana del navegador del AWS Systems Manager en la terminal de la instancia

  • 1.2. Conectese al

    sudo systemctl start cloudhsm-client

  • 1.3. Conectese al

    ./key_mgmt_util

  • 1.4.

    loginHSM -u CU -s user1 -p defaultPassword

  • 1.5.

    genSymKey -t 31 -s 32 -l aes256

  • 1.6.

    genRSAKeyPair -m 2048 -e 65541 -l rsa_test

  • 1.7.

    genECCKeyPair -i 12 -l ecc12

  • 1.8.

    exit

2. Importando y exportando llaves con el CloudHSM

(detalles)
En este modulo "you will be using key_mgmt_util software provided in cloudHSM Client package to import symmetric keys, export symmetric keys and wrap keys."

  • 2.1. Vuelva a la pestaña o ventana del navegador del AWS Systems Manager en la terminal de la instancia

  • 2.2. The first command uses OpenSSL to generate a random 256-bit AES symmetric key. It saves the key in the aes256.key file.

    openssl rand -out aes256-forImport.key 32

  • 2.3. Conectese al

    ./key_mgmt_util

  • 2.4.

    loginHSM -u CU -s user1 -p defaultPassword

  • 2.5. Generate a wrapping key on the HSM and take note of the key handle returned.You will reuire it in the below exercises.

    genSymKey -t 31 -s 32 -l wrappinhKey

  • 2.6. Use the imSymKey to import the AES key from the aes256.key file into the HSMs.

    imSymKey -f aes256.key -w -t 31 -l imported

  • 2.7. Generate a 3DES key and note the key handle returned. This key will be exported later

    genSymKey -t 21 -s 24 -l 3DES_export

  • 2.8. This command exports a Triple DES (3DES) symmetric key (key handle -k). It uses an existing AES key (key handle -w) in the HSM as the wrapping key. Then it writes the plaintext of the 3DES key to the 3DES.key file.

    exSymKey -k -w -out 3DES.key

  • 2.9.

    openssl enc -d -aes-256-cbc -in plain.txt -out encr.txt -pass file:./3DES.key
